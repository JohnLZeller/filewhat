A .jsbeautifyrc file is used to configure the use of js-beautify, which is a JS beautifier available via npm that reformats and reindents bookmarklets, ugly JavaScript, unpack scripts packed by the JS compressor 'packer', as well as deobfuscate scripts processed by javascriptobfuscator.com.
