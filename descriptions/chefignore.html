The chefignore file is used by Chef to tell knife which cookbook files in the chef-repo should be ignored when uploading data to the Chef server. Chef is a powerful automation platform that transforms infrastructure into code by automating how infrastructure is configured, deployed, and managed across your network, no matter its size. knife is a command-line tool that provides an interface between a local chef-repo and the Chef server.
